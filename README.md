# 🦞 小龙虾 Android App - 快速获取APK

## ⚡ 3步获取可安装APK（最简单的方式）

### 第1步：获取GitHub Token
访问：https://github.com/settings/tokens → 创建token（选择repo权限）→ 复制token

### 第2步：运行命令
```bash
cd /Users/adrianchen/WorkBuddy/20260310235638/crayfish_app
export GITHUB_TOKEN=你的token
./push_to_github.sh
```

按提示输入你的GitHub用户名，仓库名称可以直接回车使用默认值。

### 第3步：下载APK
脚本会显示Actions链接，点击后等待2-3分钟，在Artifacts中下载APK即可安装。
